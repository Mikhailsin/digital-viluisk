# 🏙️ Цифровой двойник Вилюйска

Интерактивная веб-платформа для мониторинга городской инфраструктуры, туризма и ЖКХ города Вилюйска (Республика Саха, Якутия).

## 🛠️ Стек технологий

### Фронтенд

| Технология              | Назначение                      |
| ----------------------- | ------------------------------- |
| React + Vite            | UI фреймворк                    |
| Leaflet + React-Leaflet | Интерактивная карта             |
| CartoDB Dark            | Тёмные тайлы карты (без токена) |
| Recharts                | Графики хронологии датчиков     |

| @react-three/drei       | Хелперы для Three.js            |

### Бэкенд

| Технология | Назначение      |
| ---------- | --------------- |
| FastAPI    | REST API сервер |
| Uvicorn    | ASGI сервер     |
| Python     | Язык бэкенда    |

| Формат           | Назначение                   |
| ---------------- | ---------------------------- |

## 🚀 Запуск

### Требования

- Python 3.8+
- Node.js 18+
- npm

### Бэкенд

```bash
cd backend
pip install fastapi uvicorn
uvicorn main:app --reload
```

Сервер запустится на `http://localhost:8000`

Swagger документация: `http://localhost:8000/docs`

### Фронтенд

```bash
cd frontend
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`

## 🎯 Проблема и решение

### Проблема

- Нет единой цифровой карты Вилюйска
- Администрация не видит состояние ЖКХ в реальном времени
- Туристы не знают что посмотреть в городе
- Нет интерактивной навигации по городу

### Решение

Веб-приложение **Цифровой двойник Вилюйска** — интерактивная карта города с:

- 📍 **29 реальных объектов** города из OpenStreetMap
- 🏛️ **6 ключевых объектов** с 3D моделями (администрация, музей, больница, школа, набережная, дворец культуры)
- 📊 **6 датчиков ЖКХ** с обновлением в реальном времени (каждые 5 секунд)
- 📈 **Хронология** показаний датчиков за 24 часа
- 🗂️ **Фильтрация** объектов по категориям (туризм, инфраструктура, транспорт, образование)
